Glossary

Done:

- *.so extension: (Shared Object) Shared library files for C and C++. They are linked during the runtime [repo/.gitignore]
- Bitrate: Metric, number of bits that are conveyed or processed per unit of time. For streams, it ranges between 500 - 8000 kbps [L-02]
- FATE: FFmpeg Automated Testing Environment
- Filtergraph: Directed graph of connected filters. It can contain cycles, and there can be multiple links between a pair of filters. Sequence of filterchains. A filter with no input pads is called a "source", and a filter with no output pads is called a "sink". [L-02]
- SDL: Simple DirectMedia Library. https://www.libsdl.org/ [L-01]
- vpath: A utility in Makefile to specify location of dependencies. https://www.cmcrossroads.com/article/basics-vpath-and-vpath [Makefile]
